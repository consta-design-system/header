# Header для дизайн-системы Consta

Здесь можно посмотреть документацию и примеры компонента Table. Компонент взят из библиотеки [rc-table](https://github.com/react-component/table) и адаптирован для [дизайн-системы Consta](https://consta.gazprom-neft.ru/).

[Посмотреть документацию и примеры](?path=/docs/components-table--playground)

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/header` или `npm install @consta/uikit @consta/header`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/header` были вложены в компонент `Theme`.

#### Всë!
