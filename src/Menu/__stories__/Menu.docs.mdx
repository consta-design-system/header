## Пункты меню

За пункты меню отвечает свойство `items` (тип [DefaultItem](#свойства)). В него входит:

- `label` — текст пункта (обязательный),
- `href` — ссылка, по которой с него можно перейти,
- `active` — отмечает активный пункт меню,
- `onClick` — возвращает клик на этот пункт,
- `subMenu` — массив с описанием подпунктов.

##### Пример обычного меню

```tsx
import { Menu } from '@consta/header/Menu'

const items = [
  {
    label: 'Пункт 1',
    href: '#',
    onClick: e => {
      e.preventDefault()
      e.stopPropagation()
    },
    subMenu: [{ label: 'Пункт 1-1' }, { label: 'Пункт 1-2' }],
  },
  { label: 'Пункт меню 2' },
  { label: 'Пункт меню 3' },
]

export const MenuExampleBasic = () => <Menu items={items} />
```

##### Пример меню с кастомным типом

Вы можете сделать меню с кастомным типом данных, отличающимся от [DefaultItem](#свойства).
В этом случае используйте геттеры
`getItemHref`, `getItemLabel`, `getItemActive`, `getItemOnClick`, `getItemSubMenu`,
чтобы компонент понимал, откуда брать данные.

```tsx
import { Menu } from '@consta/header/Menu'

const menu: MenuItem[] = [
  {
    name: 'Пункт меню 1',
    href: '#',
    onClick: e => {
      e.preventDefault()
      e.stopPropagation()
    },
    sub: [{ name: 'Пункт меню 1-1' }, { name: 'Пункт меню 1-2' }],
  },
  { name: 'Пункт меню 2', sub: [{ name: 'Пункт меню 2-1' }] },
  {
    name: 'Пункт меню 3',
    sub: [
      { name: 'Пункт меню 3-1', sub: [{ name: 'Пункт меню 3-1-1' }] },
      { name: 'Пункт меню 3-2', sub: [{ name: 'Пункт меню 3-2-1' }] },
      {
        name: 'Пункт меню 3-3',
        sub: [
          { name: 'Пункт меню 3-1', sub: [{ name: 'Пункт меню 3-1-1' }] },
          { name: 'Пункт меню 3-2', sub: [{ name: 'Пункт меню 3-2-1' }] },
          {
            name: 'Пункт меню 3-3',
            sub: [
              { name: 'Пункт меню 3-3-1', sub: [{ name: 'Пункт меню 3-3-1-1' }] },
              { name: 'Пункт меню 3-3-2', sub: [{ name: 'Пункт меню 3-3-2-1' }] },
              { name: 'Пункт меню 3-3-3', sub: [{ name: 'Пункт меню 3-3-3-1' }] },
            ],
          },
        ],
      },
    ],
  },
]

export const MenuExampleCustom = () => (
  <Menu items={menu} getItemLabel={item => item.name} getItemSubMenu={item => item.sub} />
)
```

## Свойства

```tsx
type DefaultItem = {
  label: string
  href?: string
  target?: string
  active?: boolean
  onClick?: React.EventHandler<React.MouseEvent>
  subMenu?: DefaultItem[]
}

type MenuPropGetItemLabel<ITEM> = (item: ITEM) => string
type MenuPropGetItemHref<ITEM> = (item: ITEM) => string | undefined
type MenuPropGetItemTarget<ITEM> = (item: ITEM) => string | undefined
type MenuPropGetItemActive<ITEM> = (item: ITEM) => boolean | undefined
type MenuPropGetItemSubMenu<ITEM> = (item: ITEM) => ITEM[] | undefined
type MenuPropGetItemOnClick<ITEM> = (item: ITEM) => React.EventHandler<React.MouseEvent> | undefined
type MenuPropOnItemClick<ITEM> = (props: { e: React.MouseEvent; item: ITEM }) => void

const defaultGetItemLabel: MenuPropGetItemLabel<DefaultItem> = item => item.label
const defaultGetItemActive: MenuPropGetItemActive<DefaultItem> = item => item.active
const defaultGetItemHref: MenuPropGetItemHref<DefaultItem> = item => item.href
const defaultGetItemOnClick: MenuPropGetItemOnClick<DefaultItem> = item => item.onClick
const defaultGetItemTarget: MenuPropGetItemTarget<DefaultItem> = item => item.target
const defaultGetItemSubMenu: MenuPropGetItemSubMenu<DefaultItem> = item => item.subMenu
```

| Свойство          | Тип                         | По умолчанию            | Описание                                       |
| ----------------- | --------------------------- | ----------------------- | ---------------------------------------------- |
| `items`           | `ITEM[]`                    | -                       | Элементы меню                                  |
| `onItemClick?`    | `MenuPropOnItemClick`       | -                       | Функция клика по элементу                      |
| `getItemTarget?`  | `MenuPropGetItemTarget`     | `defaultGetItemTarget`  | Функция для определения `target` элемента      |
| `getItemHref?`    | `MenuPropGetItemTarget`     | `defaultGetItemHref`    | Функция для определения `href` элемента        |
| `getItemLabel?`   | `MenuPropGetItemLabel`      | `defaultGetItemLabel`   | Функция для определения заголовка элемента     |
| `getItemActive?`  | `MenuPropGetItemActive`     | `defaultGetItemActive`  | Функция для определения активности элемента    |
| `getItemOnClick?` | `MenuPropGetItemOnClick`    | `defaultGetItemOnClick` | Функция для определения функции клика элемента |
| `getItemSubMenu?` | `MenuPropGetItemSubMenu`    | `defaultGetItemSubMenu` | Функция для определения подменю элемента       |
| `className?`      | `string`                    | -                       | Дополнительный CSS-класс                       |
| `ref?`            | `React.Ref<HTMLDivElement>` | -                       | Ссылка на корневой DOM-элемент                 |
