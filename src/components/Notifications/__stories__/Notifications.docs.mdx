# Notifications

```tsx
import { Notifications } from '@consta/header/Notifications';
```

Кнопка и обвязка для списка уведомлений [NotificationsList](?path=/docs/components-notifications--playground).

## Список уведомлений

Список уведомлений работает так же, как в компоненте [NotificationsList](?path=/docs/components-notifications--playground).

## Мобильная версия

Чтобы уведомления правильно отображалось на мобильном устройстве, добавьте `isMobile={true}`. Тогда список уведомлений откроется на весь экран.
