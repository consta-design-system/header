# Badges

```tsx
import { Badges } from '@consta/header/Badges';
```

Ряд бейджиков, красиво сворачивается, если не помещается по длине в родительский контейнер. Наследует свойства компонента [Badge](##LIBS.STAND|stand:uikit-components-badge-stable).

> Этот компонент не встраивается в шапку сам по себе, а используется в [уведомлениях](##LIBS.STAND|stand:header-components-notifications-stable). Но вы можете использовать его как-то иначе.
