## Содержимое

За данные для бейдижков отвечает свойство `items` (тип [DefaultItem](#свойства)). В него входит:

- `label` — текст на бейджике,
- `status` — статус бейджика, варианты: `'success'` , `'error'` , `'warning'`, `'normal'`, `'system'` (подрбонее см. [Badge](##LIBS.STAND|stand:uikit-components-badge-stable)),
- `icon` — любая [иконка](##LIBS.STAND|stand:uikit-components-icons-stable).

```tsx
const items = [
  {
    label: 'отчеты',
    status: 'normal',
  },
  {
    label: 'файлы',
    status: 'warning',
  },
  {
    label: 'система',
    status: 'success',
  },
  // и так далее
];

export const BadgesExampleBasic = () => <Badges items={items} />;
```

<BadgesExampleBasic />

Вы можете сделать бейджики с кастомным типом данных, отличающимся от [DefaultItem](#свойства).
В этом случае используйте геттеры `getItemLabel`, `getItemStatus`, `getItemIcon`,
чтобы компонент понимал, откуда брать данные.

## Форма

Поменять форму бейджика можно с помощью `form`. Варианты: `'default'`, `'round'`.

```tsx
<Badges items={items} form="default" />
<Badges items={items} form="round" />
```

<BadgesExampleForm />

## Размер

За размер компонента отвечает свойство `size`.

```tsx
<Badges items={items} size="l" />
<Badges items={items} size="m" />
<Badges items={items} size="s" />
```

## Внешний вид

За внешний вид компонента отвечает свойство `view`.

```tsx
<Badges items={items} view="filled" />

```

## Свойства

```tsx
type DefaultItem = {
  label: string;
  status?: BadgePropStatus;
  icon?: IconComponent;
};

type BadgesPropGetItemLabel<ITEM> = (item: ITEM) => string;
type BadgesPropGetItemStatus<ITEM> = (
  item: ITEM,
) => BadgePropStatus | undefined;
type BadgesPropGetItemIcon<ITEM> = (item: ITEM) => IconComponent | undefined;

const defaultGetItemIcon: BadgesPropGetItemIcon<DefaultItem> = (item) =>
  item.icon;
const defaultGetItemLabel: BadgesPropGetItemLabel<DefaultItem> = (item) =>
  item.label;
const defaultGetItemStatus: BadgesPropGetItemStatus<DefaultItem> = (item) =>
  item.status;
```

| Свойство                        | Тип                         | По умолчанию           | Описание                                  |
| ------------------------------- | --------------------------- | ---------------------- | ----------------------------------------- |
| [`items`](#содержимое)          | `ITEM[]`                    | -                      | Данные для бейджиков                      |
| [`form?`](#форма)               | `"default", "round"`        | `"default"`            | Форма                                     |
| [`size?`](#размер)              | `"m", "s", "l"`             | `"m"`                  | Размер                                    |
| [`view?`](#внешний-вид)         | `"filled", "stroked"`       | `"filled"`             | Вид                                       |
| [`getItemLabel?`](#содержимое)  | `BadgesPropGetItemLabel`    | `defaultGetItemLabel`  | Функция для определения названия элемента |
| [`getItemStatus?`](#содержимое) | `BadgesPropGetItemStatus`   | `defaultGetItemStatus` | Функция для определения статуса элемента  |
| [`getItemIcon?`](#содержимое)   | `BadgesPropGetItemIcon`     | `defaultGetItemIcon`   | Функция для определения иконки элемента   |
| `className?`                    | `string`                    | -                      | Дополнительный CSS-класс                  |
| `ref?`                          | `React.Ref<HTMLDivElement>` | -                      | Ссылка на корневой DOM-элемент            |
