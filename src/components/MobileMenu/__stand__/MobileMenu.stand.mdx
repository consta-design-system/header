# MobileMenu

```tsx
import { MobileMenu } from '@consta/header/MobileMenu';
```

Этот компонент состоит из вертикального меню [VerticalMenu](##LIBS.STAND|stand:header-components-verticalmenu-stable) и кнопки, по которому оно вызывается.

При нажатии на кнопку меню раскрывается на всю ширину экрана.
