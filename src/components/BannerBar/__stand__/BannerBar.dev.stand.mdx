## Содержимое

За данные для баннеров отвечает свойство `items` (тип [BannerBarDefaultItem](#свойства)). В него входит:

- `label` — заголовок баннера
- `description` — описание для баннера
- `image` — картинка
- `onClick` — событие при клике на элементе
- `as` — HTML-тег компонента
- `attributes` — атрибуты элемента

##### Пример обычных баннеров

```tsx
import React from 'react';
import { BannerBar } from '@consta/header/BannerBar';

const items = [
  {
    label: 'Особенности разведки',
    description:
      'Лицензии на пользование недрами (далее для настоящей главы - лицензия) выдаются федеральным органом исполнительной власти ...',
    image:
      'https://media.istockphoto.com/id/1322277517/photo/wild-grass-in-the-mountains-at-sunset.jpg?s=612x612&w=0&k=20&c=6mItwwFFGqKNKEAzv0mv6TaxhLN3zSE43bWmFN--J5w=',
  },
];

export const BannerBarExample = () => {
  return <BannerBar items={items} view="horizontal" />;
};
```

##### Пример баннеров с кастомным типом

```tsx
import React from 'react';
import { BannerBar } from '@consta/header/BannerBar';

type Item = {
  text: string;
  description: string;
  photo?: string;
};

const items: Item[] = [
  {
    text: 'Особенности разведки',
    description:
      'Лицензии на пользование недрами (далее для настоящей главы - лицензия) выдаются федеральным органом исполнительной власти ...',
    photo:
      'https://media.istockphoto.com/id/1322277517/photo/wild-grass-in-the-mountains-at-sunset.jpg?s=612x612&w=0&k=20&c=6mItwwFFGqKNKEAzv0mv6TaxhLN3zSE43bWmFN--J5w=',
  },
];

export const BannerBarExample = () => {
  return (
    <BannerBar
      items={items}
      getItemLabel={(item) => item.text}
      getItemImage={(item) => item.photo}
      view="horizontal"
    />
  );
};
```

## Внешний вид

За внешний вид компонента отвечает свойство `view`.

```tsx
<BannerBar items={items} view="vertical" />
<BannerBar items={items} view="horizontal" />
```

## Свойства

```tsx
export type BannerBarDefaultItem = {
  onClick?: React.MouseEventHandler;
  label: string;
  description?: string;
  image?: string;
  as?: keyof JSX.IntrinsicElements;
  attributes?: JSX.IntrinsicElements[keyof JSX.IntrinsicElements];
};

export type BannerBarPropView = 'vertical' | 'horizontal';

export type BannerBarPropGetItemLabel<ITEM> = (item: ITEM) => string;
export type BannerBarPropGetItemDescription<ITEM> = (
  item: ITEM,
) => string | undefined;
export type BannerBarPropGetItemImage<ITEM> = (
  item: ITEM,
) => string | undefined;
export type BannerBarPropGetItemOnClick<ITEM> = (
  item: ITEM,
) => React.MouseEventHandler | undefined;
export type BannerBarPropGetItemAs<ITEM> = (
  item: ITEM,
) => keyof JSX.IntrinsicElements | undefined;
export type BannerBarPropGetItemAttributes<ITEM> = (
  item: ITEM,
) => JSX.IntrinsicElements[keyof JSX.IntrinsicElements] | undefined;

export type BannerBarPropOnItemClick<ITEM> = (params: {
  e: React.MouseEvent;
  item: ITEM;
}) => void;
```

| Свойство              | Тип                               | По умолчанию                | Описание                                                                |
| --------------------- | --------------------------------- | --------------------------- | ----------------------------------------------------------------------- |
| `view?`               | `BannerBarPropView`               | `vertical`                  | Внешний вид баннеров                                                    |
| `items`               | `ITEM[]`                          | -                           | Пункты меню                                                             |
| `onItemClick?`        | `BannerBarPropOnItemClick`        | -                           | Событие по нажаьию на элемент                                           |
| `getItemLabel?`       | `BannerBarPropGetItemLabel`       | `defaultGetItemLabel`       | Функция для определения заголовка элемента                              |
| `getItemOnClick?`     | `BannerBarPropGetItemOnClick`     | `defaultGetItemOnClick`     | Функция для определения события по нажатию на элемент                   |
| `getItemDescription?` | `BannerBarPropGetItemDescription` | `defaultGetItemDescription` | Функция для определения описания элемениа                               |
| `getItemImage?`       | `BannerBarPropGetItemImage`       | `defaultGetItemImage`       | Функция для определения картинки элемента                               |
| `getItemAs?`          | `BannerBarPropGetItemAs`          | `defaultGetItemAs`          | Функция для определения тэга элемента                                   |
| `getItemAttributes?`  | `BannerBarPropGetItemAttributes`  | `defaultGetItemAttributes`  | Функция для определения атрибутов элемента                              |
| `className?`          | `string`                          | -                           | Дополнительный CSS-класс для кнопки, по которой открываются уведомления |
| `ref?`                | `React.Ref<HTMLButtonElement>`    | -                           | Ссылка на корневой DOM-элемент                                          |
