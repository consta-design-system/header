## Меню

Выпадающее меню, работает на основе `MobileMenu` или `Menu` (зависит от ширины экрана).

Добавить пункты меню можно с помощью свойства `menu`, подробности — в [документации компонента меню](##LIBS.STAND|stand:header-components-menu-stable).

## Логотип

За логотип отвечает свойство `logo`. В него можно поместить что угодно.

Чтобы поставить ссылку с логотипа, используйте `logoHref`.

## Уведомления

Этот компонент работает на основе `NotificationsList`. Как его настроить:

- данные для уведомлений передаются в `notifications`,
- действия, которые можно с этими уведомлениями совершать, можно указать в `notificationsActions`,
- заголовок для выпадающего блока с уведомлениями — `notificationsTitle`.

Чтобы сгруппировать уведомления по дням, добавьте `notificationsGroupByDay`.

[Подробнее об уведомлениях](##LIBS.STAND|stand:header-components-notificationslist-stable)

## Выпадающее меню с плиткой

Это выпадающее меню работает на основе `TileMenu`.

[Посмотреть документацию](##LIBS.STAND|stand:header-components-tilemenu-stable)

## Логин

Блок с кнопкой для входа и информацией о пользователе.

У блока есть два состояния:

- пользователь вошёл в систему — `userLogined={true}`, в блоке будет видна информация об этом пользователе,
- пользователь может войти — `userLogined={false}`, в блоке будет кнопка для входа.

```tsx
<Header userLogined={false} />
```

<HeaderExampleNotLogined />

Вы можете указать данные о пользователе:

- имя пользователя — в свойстве `userName`,
- изображение — `userAvatar`,
- дополнительную информацию — в `userInfo`.

За текст на кнопке для входа отвечает свойство `loginButtonLabel`.

```tsx
<Header
  userName="Иванов Иван"
  userInfo="Владелец"
  loginButtonLabel="Войти"
  userLogined={true}
/>
```

## Переключатель тем

С помощью переключателя тем можно поменять можно менять оформление страницы. Для этого нужно:

- подключить темы и пресеты из компонента [Theme](##LIBS.STAND|stand:uikit-components-theme-stable) (или свои),
- передать описание тем, названий и иконок для переключателя в свойстве `themeItems`,
- указать текущую тему в свойстве `theme`,
- добавить функцию для изменения темы в свойстве `onThemeChange`.

Для работы переключателя нужны:

- `key` — уникальный идентификатор темы,
- `label` — название темы,
- `icon` — [иконка](##LIBS.STAND|stand:uikit-components-icon-stable), по нажатию на которую переключается тема.

Можно использовать геттеры `getThemeTogglerItemKey`, `getThemeTogglerItemLabel`, `getThemeTogglerItemIcon`.

Переключатель работает так же, как стандартный компонент для переключения тем — [ThemeToggler](##LIBS.STAND|stand:uikit-components-themetoggler-stable).

```tsx
import {
  ThemePreset,
  presetGpnDefault,
  presetGpnDark,
  presetGpnDisplay,
} from '@consta/uikit/Theme';
import { IconLightningBolt } from '@consta/uikit/IconLightningBolt';
import { IconMoon } from '@consta/uikit/IconMoon';
import { IconSun } from '@consta/uikit/IconSun';

export type ThemeItem = {
  icon: IconComponent;
  key: string;
  label: string;
  preset: ThemePreset;
};

export const themes: ThemeItem[] = [
  {
    key: '1',
    label: 'Светлая',
    icon: IconSun,
    preset: presetGpnDefault,
  },
  {
    key: '2',
    label: 'Темная',
    icon: IconMoon,
    preset: presetGpnDark,
  },
  {
    key: '3',
    preset: presetGpnDisplay,
    label: 'Системная',
    icon: IconLightningBolt,
  },
];

export const HeaderExampleThemeToggler = () => {
  const [theme, setTheme] = useState<ThemeItem>(themes[0]);
  return (
    <Theme preset={theme.preset}>
      <Header
        logo={<ConstaLogo />}
        userName="Иванов Иван"
        userInfo="Владелец"
        loginButtonLabel="Войти"
        userLogined={true}
        notifications={notifications}
        notificationsActions={notificationsActions}
        notificationsTitle="Уведомления"
        notificationsGroupByDay={true}
        tileMenu={tileMenu}
        tileMenuTitle="Сервисы"
        style={{ zIndex: 100 }}
        themeItems={themes}
        theme={theme}
        onThemeChange={({ value }: { value: ThemeItem | null }) =>
          value && setTheme(value)
        }
      />
    </Theme>
  );
};
```

## Зафиксированная шапка

Чтобы зафиксировать шапку при прокрутке страницы у верхнего края экрана, используйте свойство `fixed`.

```tsx
<Header
  logo={<ConstaLogo />}
  userName="Иванов Иван"
  userInfo="Владелец"
  loginButtonLabel="Войти"
  userLogined={true}
  notifications={notifications}
  notificationsActions={notificationsActions}
  notificationsTitle="Уведомления"
  notificationsGroupByDay={true}
  tileMenu={tileMenu}
  tileMenuTitle="Сервисы"
  menu={menu}
  style={{ zIndex: 100 }}
  fixed
/>
```

## Хлебные крошки

Чтобы добавить под шапкой хлебные крошки, используйте `breadcrumbs`. Это свойство ведёт себя так же, как компонент [Breadcrumbs](##LIBS.STAND|stand:uikit-components-breadcrumbs-stable) из библиотеки @consta.uikit.

```tsx
type Page = {
  icon?: IconComponent;
  href: string;
  label: string;
  subMenu?: Page[];
};

export const breadcrumbs: Page[] = [
  {
    icon: IconHome,
    label: 'Главная',
    href: 'https://url.com/page-1',
    subMenu: [
      {
        label: 'Основное',
        href: 'https://url.com/page-1-1',
      },
      {
        label: 'Другое',
        href: 'https://url.com/page-1-2',
      },
    ],
  },
  {
    label: 'Раздел',
    href: 'https://url.com/page-2',
    subMenu: [
      {
        label: 'Раздел 1',
        href: 'https://url.com/page-2-1',
      },
    ],
  },
  {
    label: 'Подраздел',
    href: 'https://url.com/page-3',
  },
  {
    label: 'Элемент подраздела',
    href: 'https://url.com/page-4',
  },
  {
    label: 'Дополнительные свойства элемента подраздела',
    href: 'https://url.com/page-5',
  },
  {
    label: 'Детальное описание свойства элемента подраздела',
    href: 'https://url.com/page-6',
  },
];

export const HeaderExampleBreadcrumbs = () => (
  <Header
    // ...
    breadcrumbs={breadcrumbs}
  />
);
```

## Адаптивность

На мобильных экранах в шапке остаются только логотип или название сервиса и кнопка-гамбургер. Если нажать на кнопку, открывается боковая панель, в которую «спрятались» все остальные компоненты. Меню становится вертикальным, другие компоненты отображаются выше.

> Чтобы посмотреть, как это работает, уменьшите ширину окна браузера
