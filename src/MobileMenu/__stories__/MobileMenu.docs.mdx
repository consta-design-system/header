# Menu

## Свойства

| Свойство         | Тип                              | По умолчанию            | Описание                                                                                                                   |
| ---------------- | -------------------------------- | ----------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| `items`          | `ITEM[]`                         | -                       | Элементы (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))                                            |
| `onItemClick`    | `VerticalMenuPropOnItemClick`    | -                       | Функция клика по элементу (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))                           |
| `getItemHref`    | `VerticalMenuPropGetItemHref`    | `defaultGetItemHref`    | Функция для определения `href` элемента (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))             |
| `getItemLabel`   | `VerticalMenuPropGetItemLabel`   | `defaultGetItemLabel`   | Функция для определения заголовка элемента (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))          |
| `getItemTarget`  | `VerticalMenuPropGetItemTarget`  | `defaultGetItemTarget`  | Функция для определения `target` элемента (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))           |
| `getItemActive`  | `VerticalMenuPropGetItemActive`  | `defaultGetItemActive`  | Функция для определения статуса активности элемента (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground)) |
| `getItemOnClick` | `VerticalMenuPropGetItemOnClick` | `defaultGetItemOnClick` | Функция для определения функции клика элемента (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))      |
| `getItemSubMenu` | `VerticalMenuPropGetItemSubMenu` | `defaultGetItemSubMenu` | Функция для определения подменю элемента (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))            |
| `header`         | `React.ReactNode`                | -                       | Шапка меню (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))                                          |
| `className?`     | `string`                         | -                       | Дополнительный класс (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))                                |
| `ref?`           | `React.Ref<HTMLDivElement>`      | -                       | Ссылка на корневой DOM-элемент (см. [VerticalMenu](/?path=/docs/components-verticalmenu--playground))                      |
