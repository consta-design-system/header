Шапка (Header) — составной компонент, собирается из нескольких модулей.

## Как собрать шапку

Есть два варианта: использовать компонент `Header` или собрать кастомную шапку.

`Header` уже собран: он состоит из определённых модулей и обладает жёсткой структурой. Вы можете не показывать какой-то модуль, но не можете поменять его расположение. Этот вариант подходит, если хочется всё сделать быстро, устраивает расположение элементов и не смущает множество настроек. [Подробнее про Header](##LIBS.LIB.STAND/lib:header/stand:components-header-stable)

Чтобы собрать кастомную шапку, используйте `Layout` в качестве каркаса, в который можно поместить компоненты, элементы и модули. Это очень гибкий вариант: вы можете добавить в каркас почти что угодно и расположить так, как это нужно для проекта. [Подробнее о кастомной шапке](##LIBS.LIB.STAND/lib:header/stand:components-layout-stable)

## Начало работы

### Установите пакет и зависимости

`yarn add @consta/uikit @consta/header`

или

`npm install @consta/uikit @consta/header`

### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты были вложены в [компонент `Theme`](##LIBS.LIB.STAND/lib:uikit/stand:components-theme-stable).

### Соберите шапку

Например, так:

```js
import { Theme, presetGpnDefault } from '@consta/uikit/Theme';
import { Header } from '@consta/header/Header';

const App = () => {
  return (
    <Theme preset={presetGpnDefault}>
      <Header />
    </Theme>
  );
};
```
