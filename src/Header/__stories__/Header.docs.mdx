# Header

Готовая шапка для проекта с жёсткой структурой, состоит из нескольких модулей. Модули можно скрывать и настраивать, но поменять расположение не получится.

### Меню

Выпадающее меню, работает на основе `MobileMenu` или `Menu` (зависит от ширины экрана).

Добавить пункты меню можно с помощью свойства `menu`, подробности — в [документации компонента меню](?path=/docs/components-menu--playground).

### Логотип

За логотип отвечает свойство `logo`. В него можно поместить что угодно.

### Уведомления

Этот компонент работает на основе `NotificationsList`. Как его настроить:

- данные для уведомлений передаются в `notifications`,
- действия, которые можно с этими уведомлениями совершать, можно указать в `notificationsActions`,
- заголовок для выпадающего блока с уведомлениями — `notificationsTitle`.

Чтобы сгруппировать уведомления по дням, добавьте `notificationsGroupByDay`.

[Подробнее об уведомлениях](?path=/docs/components-notificationslist--playground)

### Выпадающее меню с плиткой

Это выпадающее меню работает на основе `TileMenu`.

[Посмотреть документацию](?path=/docs/components-tilemenu--playground)

### Логин

Блок с кнопкой для входа и информацией о пользователе.

У блока есть два состояния:

- пользователь вошёл в систему — `userLogined={true}`, в блоке будет видна информация об этом пользователе,
- пользователь может войти — `userLogined={false}`, в блоке будет кнопка для входа.

```tsx
<Header userLogined={false} />
```

<HeaderExampleNotLogined />

Вы можете указать данные о пользователе:

- имя пользователя — в свойстве `userName`,
- изображение — `userAvatar`,
- дополнительную информацию - в `userInfo`.

За текст на кнопке для входа отвечает свойство `loginButtonLabel`.

```tsx
<Header userName="Иванов Иван" userInfo="Владелец" loginButtonLabel="Войти" userLogined={true} />
```
